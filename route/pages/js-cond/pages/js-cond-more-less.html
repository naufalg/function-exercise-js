Soal

Buat conditional untuk membandingkan 2 buah nilai. Tampilkan apakah salah satu nilai misal a lebih besar dari b, a lebih kecil dari b, dan a sama dengan b.

Jawaban

                
                    let a,b;
                    if (a>b) {
                        console.log("a lebih besar dari b dengan nilai " + a);
                    } else if (a < b) {
                        console.log("b lebih besar dari a dengan nilai " + b);
                    } else if (a==b){
                        console.log("nilai a dan b sama besar")
                    } else {
                        console.log("salah input data")
                    }
