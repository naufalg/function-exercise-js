Buatlah rekursive function untuk menghitung angka faktorial
Contoh perhitungan angka faktorial:
4! = 4 * 3!
4! = 4 * 3 * 2!
4! = 4 * 3 * 2 * 1
4! = 24

Jawaban

                
                    function faktorial(angka) {
                        if (angka == 0 || angka == 1) {
                          return 1;
                        } else if (angka > 1) {
                          return angka * faktorial(angka - 1);
                        } else {
                          alert("angka harus >= 0");
                        }
                      }
