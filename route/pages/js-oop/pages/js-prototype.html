Soal

Buatlah method pada string yang dapat mengembalikan nilai string itu sendiri dalam keadaan terbalik
Contoh: thor -> roht
rusak -> kasur

Jawaban

                        
                          String.prototype.reverseString = function () {
                            var newStr = "";
                            var i = this.length;
                            while (i > 0) {
                              newStr += this.substring(i - 1, i);
                              i--;
                            }
                            return newStr;
                          };
                          
                          "Text123".reverseString(); //"321txeT"


Soal

Buatlah method pada number bernama ‘isPrima()’ yang dapat mengembalikan nilai boolean ‘true’ jika number tersebut merupakan bilangan prima
Contoh:
7.isPrima() -> true
6.isPrima() -> false

Jawaban

                    
                      Number.prototype.isPrima = function () {
                        if (this == 1) {
                          return false;
                        } else if (this == 2) {
                          return true;
                        } else {
                          for (let x = 2; x < this; x++) {
                            if (this % x == 0) {
                              return false;
                            }
                          }
                          return true;
                        }
                      };
                      
                      
                      let num1 = 11;
                      num1.isPrima(); // true
